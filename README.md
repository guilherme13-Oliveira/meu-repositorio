# Meu Primeiro Repositório no GitHub

Este repositório foi criado como parte do desafio **"Criando seu Primeiro Repositório no GitHub"** da [DIO](https://www.dio.me/).

## Objetivo
Praticar os comandos básicos do **Git** e aprender o fluxo completo de versionamento de código, desde a criação do repositório até o envio de alterações para o GitHub.

## Tecnologias Utilizadas
- Git
- GitHub
- Terminal / CMD

## Passos Realizados
1. Criação de um repositório no GitHub.  
2. Clonagem do repositório para o computador.  
3. Criação de um arquivo simples (README.md e desafio.txt).  
4. Commit das alterações.  
5. Envio (push) para o GitHub.
